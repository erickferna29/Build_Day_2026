# AgroCapital FIRA Credit App

Aplicación React + Vite para evaluar solicitudes de crédito bajo las reglas de FIRA.

## Comandos

- `npm install` - instala dependencias
- `npm run dev` - ejecuta el servidor de desarrollo
- `npm run build` - crea el build de producción

## Descripción

Interfaz simple para capturar datos de cliente, monto solicitado, plazo y parámetros financieros.
